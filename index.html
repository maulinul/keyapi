☁️ Cloud Scheduler — GitHub Actions

📋 Setup 4 Langkah
1Download api-vault-check.yml → taruh di .github/workflows/ repo GitHub.
2Repo GitHub → Settings → Secrets → Actions, tambahkan:
• VAULT_GH_TOKEN → isi dari tab Akses
• VAULT_GIST_ID → Gist ID dari tab Sync
3Commit + push → Actions berjalan otomatis sesuai jadwal, meski browser tertutup.
4Buka web app → 📂 Load from Gist untuk lihat status terbaru dari cloud.
